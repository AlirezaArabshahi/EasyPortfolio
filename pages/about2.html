About This Router

Simple SPA Router Implementation

This is a demonstration of a simple single-page application router built with vanilla JavaScript, inspired by the vanilla-spa-router repository. It provides smooth navigation between pages without full page reloads.

Vanilla JavaScript History API Dynamic Loading Clean URLs

The router is designed to be lightweight and easy to integrate into any project. It follows modern web development practices while maintaining simplicity.

Key features include: • Clean URL routing without hash fragments
• Dynamic content loading
• Active navigation state management
• Error handling and 404 pages
• Browser back/forward button support
